# 📊 Europe Sales Records Analysis & Dashboard

## 🎯 Objective
The primary goal of this project is to perform Exploratory Data Analysis (EDA) on European sales data. This analysis uncovers key business insights, sales trends, and profitability metrics across different regions and item types to support data-driven decision-making.

## 🛠️ Tools & Technologies Used
* **Microsoft Excel:** Data Cleaning, Advanced Formulas, Pivot Tables, and Interactive Dashboard Creation.

## 📈 Key Insights & Findings
* Identified the most profitable item types and performing regions.
* Analyzed total revenue, total cost, and total profit trends across different order priorities.
* Evaluated the impact of sales channels (Online vs. Offline) on overall profitability.
